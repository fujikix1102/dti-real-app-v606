# Local embedded payload viewer boundary

This selection ledger prepares a local-only embedded viewer.

Allowed:
- show payload identity
- show chain summary tables
- show diagnostics
- show MAP / bestfit candidate rows
- show parameter identity
- show thin sample previews
- show source identity
- show explicit boundary warnings
- compare payload readouts as historical-chain inventory

Not allowed:
- run new MCMC
- run CLASS
- run likelihood
- copy full raw chains into viewer package
- make a new posterior claim
- claim Planck/CMB validation
- claim DESI fit
- claim physics validation
- claim Hubble tension solution
- claim physical mechanism proof
- update manuscript
- update public app
- git add/commit/push

Interpretation:
This is a viewer-selection gate only.
It raises audit usability, not physics-proof status.
